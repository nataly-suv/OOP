поля в планере:
0. ID 
1. дата создания
2. ФИО автора 
3. Дед лайн 
4. Задача
5. Приоритет

Логика: 
4 класса:
1) PrintPlaner 
    отвечает за форматирование файла csv и вывод в консоль планера 
2) PrintTaskInPlaner
    отвечает за вывод в консоль задач определенного уровня, по запросу от пользователя 
3) AddTask 
    отвечает за добывление в csv файл новой задачи
    автоматически прописываются ID и дата создания
4) Talking 
    отвечает за диалог с пользователем
